## ✅ PRE-SUBMISSION CHECKLIST

Before you hit submit, verify:

### Manuscript Content
- [ ] Abstract ≤150 words, self-contained
- [ ] Introduction establishes problem AND gap in models
- [ ] Results shows 5+ major findings with figures
- [ ] Methods is reproducible
- [ ] Discussion addresses significance & limitations
- [ ] References in Nature format (~80-100)

### Figures (Max 7)
- [ ] All ≥300 dpi resolution
- [ ] Colorblind-accessible color schemes
- [ ] Detailed captions (150+ words each)
- [ ] Panels labeled (a, b, c)
- [ ] Files organized and named

### Submission Materials
- [ ] Cover letter (customize from template)
- [ ] Data availability statement (GitHub link)
- [ ] 5-6 suggested reviewers (no conflicts)
- [ ] Conflict of interest declaration
- [ ] Supplementary materials organized

### Technical
- [ ] No tracked changes in final version
- [ ] Line numbers present
- [ ] Double-spaced for review
- [ ] Spell-checked and proofread
- [ ] All citations verified
