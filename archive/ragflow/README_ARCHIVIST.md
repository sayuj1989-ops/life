# RAGFlow (Vendored)

This directory contains a full vendored copy of the [RAGFlow](https://github.com/infiniflow/ragflow) project.

**Status:** Archived / Experimental Tool.

## Purpose
This tool was investigated for use in generating RAG-based insights from the research literature and manuscript drafts. It is not currently part of the active production pipeline (CI/CD) but is preserved here for local experimental use if needed.

## Integration
Previous integration scripts (e.g., `ragflow_client.py`, `ragflow_manuscript_analyzer.py`) have been moved to `docs/archive/` or `archive/` folders. They relied on running this RAGFlow instance via Docker.

## Usage
To use this tool, refer to the original documentation within this directory or the experimental notes in `docs/archive/`.
