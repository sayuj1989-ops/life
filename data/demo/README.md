# Demo Data

`phase2_synthetic_longitudinal.csv` is a deterministic synthetic cohort generated by `scripts/validation/longitudinal_progression_validation.py --generate-demo`.

Purpose:

- verify the Phase 2 longitudinal schema;
- exercise progression label creation;
- run model-comparison plumbing without private clinical data.

This file is not clinical evidence and must not be interpreted as a real AIS cohort.
