# Documentation Directory

Research documentation, integration guides, and historical archives.

## Structure

### `notes/`
Integration guides, setup documentation, and research notes that may be useful for future work:
- Platform setup guides (RunPod, SkyPilot, etc.)
- Integration documentation (Instella, RAGflow, etc.)
- Research frameworks (Longevity, etc.)
- Quick start guides

**Files:** 18 documents

### `archive/`
Historical files that document the research process but are not needed for reproduction:
- Manuscript drafts and planning documents
- Peer review reports and responses
- Analysis scripts (superseded by current pipeline)
- Research summaries
- Deprecated tools and clients

**Files:** 57+ documents

## Usage

These files are kept for:
1. **Historical record** - Track project evolution
2. **Context** - Understand design decisions
3. **Learning** - Reference for future projects

## For New Contributors

Start with:
1. Main [README.md](../README.md) for project overview
2. [CITATION.cff](../CITATION.cff) for citation info
3. [manuscript/](../manuscript/) for paper content
4. `notes/ALPHAFOLD_RESEARCH_BUILD_COMPLETE.md` for analysis pipeline status

Archive files can be safely ignored for reproduction.
