# Spinal Sleep Spindles: Simulation Report

## Overview
This simulation tests the 'Spinal Sleep Spindles' hypothesis, which posits that night-time recumbency (unloading) allows the spine to run an 'Latent Imagination' active inference subroutine to optimize its proprioceptive derivative gain ($K_d$).

## Results
- **Normal Sleep**: $K_d$ is successfully replenished each night, maintaining a stable, over-damped controller.
- **Sleep Deprivation**: The off-line optimization window is too short. Daytime degradation outpaces nightly recovery, plunging the system into the 'Derivative Gain Trap'. Once $K_d$ drops below the critical damping threshold, the mechanical system experiences runaway postural error (representing scoliotic buckling/instability).

## Conclusion
Circadian desynchronization or sleep deprivation during the rapid adolescent growth phase may mechanically destabilize the spine by failing to optimize the active inference predictive model.
