## BACKUP PLANS

### If Nature Desk-Rejects
**What to do:**
1. Request feedback from Nature editor
2. Revise based on feedback
3. Submit to: **Nature Biomedical Engineering** or **eLife**

### If Nature Sends Major Revisions
**Assess:**
- Can you realistically address all concerns?
- Would 6-8 weeks revisions be feasible?
- OR submit to eLife (faster, equally rigorous)?

### Target Journals (in order)
1. **Nature** (primary)
2. **Nature Biomedical Engineering** (if applied focus needed)
3. **eLife** (open access, very rigorous)
4. **PLOS Biology** (open access, interdisciplinary)
5. **Current Biology** (faster review)
6. **Developmental Cell** (developmental focus)

## SUCCESS FACTORS

### Strengths to Emphasize
✓ Novel theoretical framework (Information-Elasticity Coupling)
✓ Bridges physics, biology, medicine in meaningful way
✓ Testable predictions (especially microgravity)
✓ Cross-species validation
✓ Potential clinical applications
✓ Open-source code + reproducible

### Potential Reviewer Concerns
1. **"Isn't this just Cosserat mechanics?"**
   - Response: No, Cosserat describes mechanical behavior; we show how developmental information couples to elasticity, a new regime.

2. **"How do you experimentally validate information-elasticity coupling?"**
   - Response: Multiple lines: comparative morphology, mutation correlations, microgravity predictions.

3. **"What's the biological mechanism?"**
   - Response: HOX genes → Morphogen gradients → Protein composition changes → Local elasticity modulation.

4. **"Scoliosis is multifactorial—why single-mechanism explanation?"**
   - Response: We provide the normal reference state; pathologies are deviations from this program.
